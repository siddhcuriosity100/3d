# ENDO Villa Landscape Monitoring Dashboard

A real-time monitoring dashboard for 50 villa plots — tracking water tank levels, soil moisture, and daily water consumption per plot.

**Live Demo:** [https://siddhcuriosity100.github.io/3d/](https://siddhcuriosity100.github.io/3d/)

---

## Features

- **Dashboard** — Summary metrics + 50 individual plot cards with tank level, soil moisture, and usage
- **Plot Layout Map** — 10 x 5 colour-coded grid showing tank status at a glance
- **Water Consumption** — Bar chart + sortable table with daily, weekly, and monthly estimates (1,200–2,000 L/plot/day)
- **Alerts** — Real-time alerts for critically low tanks, dry soil, and high water usage
- **Dark / Light Theme** — Toggle switch with saved preference
- **Auto-Refresh** — Data updates every 30 seconds
- **Search & Filter** — Find plots by name/number, filter by status, sort by any metric

## Tech Stack

- Pure HTML, CSS, JavaScript (no frameworks)
- CSS Variables for theming
- Inter font (Google Fonts)
- Hosted on GitHub Pages
